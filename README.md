Instale os pacotes necessarios dentro de um venv

python3 -m venv venv

# windows
venv\Scripts\activate.bat

# Linux
source venv/bin/activate

pip install -r requirements.txt

Depois disso, é para funcionar executando o script normalmente
(no diretorio do auto_clicker, usa python auto_clicker.py)
