
# Auto Clicker

## 1. Criar e ativar o ambiente virtual (venv)

### Windows

```bash
python3 -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

### Linux

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

---

## 2. Executar o projeto

Após instalar as dependências, execute o script normalmente dentro do diretório `auto_clicker`:

```bash
python auto_clicker.py
```

### Alternativa (Windows)

Você também pode executar diretamente:

```bash
rodar.bat
```

---

Depois disso, o programa deve funcionar corretamente.
